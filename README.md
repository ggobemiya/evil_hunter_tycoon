## GitHub Pages 업데이트 요약

이 프로젝트를 GitHub Pages (ggobemiya.github.io/evil_hunter_tycoon/)에 업데이트하려면 다음 단계를 따르세요. 자세한 내용은 `GEMINI.md`를 참조하십시오.

1.  **프로젝트 빌드**: 최신 변경사항을 반영하기 위해 프로젝트를 빌드합니다.
    ```bash
    npm run build
    ```
2.  **변경사항 커밋**: 빌드된 파일과 소스 코드 변경사항을 Git 저장소에 커밋합니다.
    ```bash
    git add -A
    git commit -m "feat: <변경 내용 요약>"
    ```
3.  **GitHub에 푸시**: 커밋된 변경사항을 원격 GitHub 저장소에 푸시합니다.
    ```bash
    git push
    ```
